Introduction to JavaScript

JavaScript is a programming language that enables interactive web pages. It is an interpreted language, meaning it runs without being compiled first.

JavaScript is primarily used for client-side web development, allowing developers to create dynamic and interactive user interfaces. It can also be used on the server-side with environments like Node.js.

Some key features of JavaScript include:

  • Dynamic typing
  • First-class functions
  • Event-driven programming
  • Asynchronous programming with Promises and async/await

JavaScript is supported by all modern web browsers, making it an essential tool for web developers.

Versions

JavaScript has evolved over the years with several versions being released. The most significant update came with ECMAScript 6 (ES6) in 2015, which introduced many new features such as arrow functions, classes, template literals, and more.

Basic Syntax

JavaScript syntax is the set of rules that define how JavaScript code is written and interpreted. Here are some basic syntax elements:

  • Variables: Used to store data values. Declared using var, let, or const. 
    let name = "John";
  • Data Types: Common data types include Number, String, Boolean, Object, and Array. 
    let age = 30; // Number
  • Functions: Blocks of code designed to perform a particular task. 
    function greet() {   
    console.log("Hello, World!");
}
  • Control Structures: Includes conditional statements and loops. 
    if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
  • Events: Code that runs in response to user actions or other occurrences. 
    button.addEventListener("click", function() {
    alert("Button clicked!");
});
  • Comments: Used to explain code and are ignored by the interpreter. 
    // This is a single-line comment
/* This is a multi-line comment */

Introduction to Script Tag

The <script> tag is used to embed or reference JavaScript code in an HTML document.

<script>
    // JavaScript code goes here
</script>

Three ways to include JavaScript

  • Internal JavaScript: Within the same HTML file but in a separate <script> tag. 
    <html>
    <head>
        <script>
            function greet() {
                alert("Hello, World!");
            }
        </script>
    </head>
    <body>
        <button onclick="greet()">Click Me</button>
    </body>
</html>
  • External JavaScript: In a separate .js file linked to the HTML file. 
    <html>
    <head>
        <script src="script.js"></script>
    </head>
    <body>
        <button onclick="greet()">Click Me</button>
    </body>
</html>
  • Body of the page: The <script> tag can also be placed just before the closing </body> tag to ensure that the HTML content is loaded before the JavaScript runs. 
    <html>
    <head>
        <title>My Page</title>  
    </head>
    <body>
        <h1>Welcome to My Page</h1>
        <button onclick="greet()">Click Me</button>
        <script>
            function greet() {
                alert("Hello, World!");
            }
        </script>
    </body>
</html>

JavaScript Rules

  • Case Sensitivity: JavaScript is case-sensitive. Variables named myVar and myvar are different.
  • Semicolons: While not required, semicolons are recommended to end statements.
  • Comments: Use comments to explain code and improve readability.
  • Variable Naming: Variable names must start with a letter, underscore (_), or dollar sign ($). They cannot start with a number and cannot contain spaces.
  • Reserved Words: Avoid using JavaScript reserved words (like if, for, while, etc.) as variable or function names.
  • Data Types: Be aware of JavaScript's dynamic typing and type coercion rules.
  • Function Declarations: Function declarations are hoisted, meaning they can be called before they are defined in the code.
  • White Spaces: JavaScript ignores extra white spaces, but it's good practice to use them for readability.

Common Errors

  • Uncaught ReferenceError: This error occurs when a variable is referenced but not defined.
  • Uncaught TypeError: This error occurs when a value is not of the expected type.
  • Uncaught SyntaxError: This error occurs when the JavaScript parser encounters invalid syntax.

Permitive Data Types

JavaScript has several data types:

  • Number: Represents both integers and floating-point numbers.
  • String: Represents a sequence of characters.
  • Boolean: Represents true or false values.
  • Undefined: Represents a variable that has been declared but not assigned a value.
  • Null: Represents an intentional absence of any object value.
  • Object: A collection of properties and methods.

Variables

Variables are used to store data values. In JavaScript, you can declare variables using var, let, or const. 

let age = 25; // Number
let name = "Alice"; // String
let isStudent = true; // Boolean

Variable Scope: Variables declared with var have function scope, while variables declared with let and const have block scope.

Global and Local Scope: Variables declared outside any function are global, while those declared inside a function are local.

Operators

JavaScript supports various operators:

  • Arithmetic Operators: +, -, *, /, %, **

    • Explanation with example 

    let a = 10;
let b = 5;  
let sum = a + b; // Addition
let difference = a - b; // Subtraction
let product = a * b; // Multiplication
let quotient = a / b; // Division
let remainder = a % b; // Modulus
let power = a ** b; // Exponentiation
  • Comparison Operators: ==, ===, !=, !==, >, <,>=, <=< /li>

    Explanation with example 

    let x = 10;
let y = "10";
let isEqual = (x == y); // true (loose equality)
let isStrictEqual = (x === y); // false (strict equality)
let isNotEqual = (x != y); // false
let isGreater = (x > 5); // true
let isLessOrEqual = (x <= 10); // true
  • Logical Operators: &&, ||, !

    • Explanation with example 

    let a = true;
let b = false;
let andResult = a && b; // false
let orResult = a || b; // true
let notA = !a; // false
  • Increment and Decrement Operators: ++, --

    • Explanation with example 

    let count = 0;
count++; // Increment count by 1
count--; // Decrement count by 1
  • Assignment Operators: =, +=, -=, *=, /=

    • Explanation with example 

    let num = 10;
num += 5; // num = num + 5
num -= 3; // num = num - 3
num *= 2; // num = num * 2
num /= 4; // num = num / 4

Switch Case

JavaScript supports the switch statement for multiple conditional checks:

  • Switch Statement: Evaluates an expression and matches it against several cases.

    • Explanation with example 

    let day = "Monday";
switch (day) {
    case "Monday":
        console.log("It's Monday.");
        break;
    case "Tuesday":
        console.log("It's Tuesday.");
        break;
    default:
        console.log("It's another day.");
}

    Complete code for practice 

    <!DOCTYPE html>
<html>
<head>
    <title>JavaScript Switch Statement Example</title>
</head>
<body>
    <script>
        let day = "Monday";
        switch (day) {
            case "Monday":
                console.log("It's Monday.");
                break;
            case "Tuesday":
                console.log("It's Tuesday.");
                break;
            default:
                console.log("It's another day.");
        }
    </script>
</body>
</html>
